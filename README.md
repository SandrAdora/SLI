# Sign Language Interpreter (SLI)

This is a medical web-base Software aimed to be a communication platform for deaf inclusion.
**!Note: If your goal is to work or/and iprove this project. Feel free to read this readme file**
Here is a workflow on how to use this project effectively. Cloning this project will automatically insall all packages need to effectively work on this project.**(still working on this project)**

# 1. Prerequites

* You have an **IDE** (VSC, VS etc)
* [python is installed](https://www.python.org/downloads/).
* [pip is installed](https://pip.pypa.io/en/stable/installation/).

# 2. Open your IDE (for windows)

* If you use VSC open it and open the terminal. Type in following command: ```bash python -m venv <name of your virtual environment>```
* open the virt. environment: ```bash <name of your virtual environment>\Scripts\activate```
